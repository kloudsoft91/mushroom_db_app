# Mushroom DB App

This project was the result of a Capstone Project run through Massey University. It is a webapp designed using NuxtJS to easily identify the different variety of mushrooms and fungi that grow in Aotearoa New Zealand.

See the companion app in the following repo: https://github.com/kloudsoft91/mushroom_db_app_admin_gui/

## Run Locally

Clone the project

```bash
  git clone https://github.com/kloudsoft91/mushroom_db_app.git
```

Go to the project directory

```bash
  cd mushroom_db_app
```

### Setup

Make sure to install the dependencies:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install
```

### Development Server

Start the development server on `http://localhost:3000`:

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev
```

### Production

Build the application for production:

```bash
# npm
npm run build

# pnpm
pnpm run build

# yarn
yarn build
```

Locally preview production build:

```bash
# npm
npm run preview

# pnpm
pnpm run preview

# yarn
yarn preview
```

## Running Tests

To run tests, run the following command

```bash
  npm run test
```
